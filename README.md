# 4 Servo Motors Control 🎮

This project demonstrates how to control four servo motors using an Arduino Uno and the Servo library.

The motors perform a sweep motion from 0° to 180° and back, then stop at 90°.

---

##  Components Used
- Arduino Uno
- 4 Servo Motors
- Jumper Wires

---

## 🔌 Connections
- Servo 1 → Pin 3
- Servo 2 → Pin 5
- Servo 3 → Pin 6
- Servo 4 → Pin 9

---

## ⚙️ How It Works
All servo motors move together in a sweep motion using a loop.  
After completing the movement, they stop at 90 degrees.
